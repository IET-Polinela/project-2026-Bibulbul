About This Project

Tentang Bogor SmartCity

Aplikasi portal aspirasi warga berbasis digital yang menghubungkan masyarakat Kota Bogor dengan pemerintah daerah secara transparan, cepat, dan akuntabel.

Tujuan Aplikasi

Menyediakan platform pelaporan masalah kota yang mudah diakses, memungkinkan warga melaporkan masalah infrastruktur, kebersihan, keamanan, dan berbagai isu lainnya secara langsung kepada pihak berwenang.

Teknologi

Dibangun menggunakan Django dan Django REST Framework sebagai backend, Bootstrap 5 untuk antarmuka responsif, serta Chart.js untuk visualisasi data statistik laporan secara real-time.

Konteks Akademik

Proyek ini merupakan tugas mata kuliah PIE 1416 Teknologi Rekayasa Internet. Dikembangkan sebagai implementasi nyata dari konsep Smart City dalam pengelolaan aspirasi dan laporan warga.

Alur Penanganan

Setiap laporan melewati alur: Reported → Verified → In Progress → Resolved, memastikan setiap masalah ditangani secara sistematis.

Tech Stack
Django Django REST Framework Bootstrap 5 Chart.js Bootstrap Icons SQLite Python 3
